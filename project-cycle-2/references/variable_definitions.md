EverCigaretteUse
Variable Name：EverCigaretteUse

What the Variable Measures：This variable measures whether a student has ever tried cigarette smoking.

Valid Codes Used： According to the project instructions: • code 1 = success • code 2 = failure

Missing or Invalid Values： Any values other than 1 or 2 will be treated as missing or invalid and will be excluded from the analysis.

Final Sample Size： The final sample size will be the number of observations with valid values after removing missing or invalid data.
BMIPCT
Variable Name：BMIPCT

What the Variable Measures：This variable measures the BMI percentile of a student.

Valid codes Used：Valid codes are numeric BMI percentile values.

Missing or Invalid Values：Missing, non-numeric, or invalid values will be removed before analysis.

Final Sample Size： The final sample size will be the number of valid BMIPCT observations used in the analysis.
